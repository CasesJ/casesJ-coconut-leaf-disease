## 🌍 Reading Location Names

### **What is "Panabo, Davao del Norte"?**

```
Panabo          = Village/Town/City (where the tree is)
Davao del Norte = Province/State (like "the North part")
```

**Easy way to remember:**
- Left part (Panabo) = Your neighborhood
- Right part (Davao del Norte) = Your region/province

## 📍 Understanding Color Dots

### **Detection Colors**

| Color | Meaning | What to Do |
|-------|---------|-----------|
| 🔴 RED | Disease Found | Spray fungicide immediately |
| 🟡 YELLOW | Warning Zone | Monitor closely |
| 🟢 GREEN | Healthy Tree | No action needed |
| ⚪ GRAY | Old Detection | Already handled? |

---

## 📊 What Do These Numbers Mean?

### **Confidence Percentage**
```
95% confidence = Almost 100% sure it's a disease
84% confidence = Pretty sure it's a disease
50% confidence = Might be, might not be (watch it)
```

**Simple rule:**
- **> 80%** = Definitely needs treatment
- **60-80%** = Probably needs treatment  
- **< 60%** = Wait and see (might be dust)

## ⏰ Time Information


## ❓ FAQ

### **Q: What if it says "Loading location..."?**
A: The system is finding the place name. Wait 1-2 seconds. It will update.

### **Q: What if it shows numbers like "7.3075, 125.6851"?**
A: That's the backup system (when place name not found). Tell the tech person about this location.

### **Q: Can I see old detections?**
A: Yes! Scroll down in Detection Log. Click any old one to see it on the map again.

### **Q: What if GPS is wrong?**
A: The map might be off by 20-50 meters. Look around the location shown. See any sick trees? Investigate.

### **Q: Do I need internet for directions?**
A: YES. "Get Directions" needs internet. Download offline maps first (ask tech team).

## 🚨 Emergency Workflow

**If many diseased trees detected:**

1. **Look at Detection Log** → See all locations
2. **Plan route** → Start with closest one? Or worst one?
3. **Click each "Directions"** → Navigate one by one
4. **Treat oldest first** → They're worse

## 📋 Checklist Before Starting

- [ ] Phone has internet (for Directions)
- [ ] Google Maps app installed (optional, but faster)
- [ ] Fungicide spray ready
- [ ] Spray equipment charged
- [ ] Bucket of water for spray mixture
- [ ] Protection on (gloves, mask)
- [ ] Notebook to record treated trees
- [ ] Camera to photo treated trees (proof of work)

---

## 🌳 After Finding a Diseased Tree

1. **Take a photo** (proof it was there)
2. **Apply treatment** (follow fungicide instructions)
3. **Wait 48 hours** (let it work)
4. **Take another photo** (see if improving)
5. **Report to supervisor** (mark as handled)

---

## 💡 Tips for Success

✅ **DO:**
- Treat trees as soon as detected (key to stopping spread)
- Follow Google Maps exactly (it knows the roads)
- Wear protection when spraying
- Keep phone charged (bring power bank)
- Take photos of treated trees (track your work)

❌ **DON'T:**
- Ignore "Loading..." status (wait for it to finish)
- Trust location 100% (be within 50m, look around)
- Spray random trees (follow directions exactly)
- Forget to water spray (mix fungicide correctly)
- Ignore old detections (could get worse)

## 📞 Help & Support

**If something doesn't work:**

1. **No location showing?**
   - Wait 2 seconds (still loading)
   - Turn volume up (might be speaking)
   - Check internet connection

2. **Directions won't open?**
   - Install Google Maps app
   - Try again with better internet
   - Copy the pin location number (coordinates)

3. **Map zooming wrong place?**
   - GPS might be off (50m is normal)
   - Look around the shown spot
   - Walk 50-100 meters to find tree

4. **Unsure if tree is diseased?**
   - Take a photo
   - Send to supervisor
   - Compare confidence % (confidence > 80% = treat it)
